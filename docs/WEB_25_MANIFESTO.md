# Web 2.5 Manifesto — Why Axie Duel is the Future of Web3 Gaming

## The problem with Web3 games today

Web3 games today fall into one of two extremes, and both lose players:

### Extreme 1 — "Web3 first" (ledger-day-zero)
- **Examples**: most P2E games of 2021-2023.
- **Onboarding**: download a wallet → bridge tokens → buy starter NFT → KYC → play.
- **Friction**: 30+ minutes from "I heard about this game" to "first match".
- **Result**: 95% drop-off before first session. Only crypto natives survive.

### Extreme 2 — "Web2 with cosmetic NFTs"
- **Examples**: Hearthstone with NFT skins, EA's "blockchain experiments".
- **Onboarding**: standard email login. NFTs are decorative, no gameplay impact.
- **Friction**: low onboarding, but NFT holders feel cheated — there's no real utility.
- **Result**: Web3 audience disengaged; the game becomes a Web2 game with a token tax.

## The Web 2.5 thesis

**Axie Duel is built for both audiences simultaneously, with no compromise on either side.**

| Layer | Web2 player journey | Web3 player journey |
|---|---|---|
| **Onboarding** | Sign in with Google in 5 seconds | Same Google sign-in (no wallet required day 1) |
| **First match** | Pick a competitive starter deck → play vs bot | Same path — wallet linking is optional, post-onboarding |
| **Progression** | Earn Dust (off-chain currency) → buy packs → build collection | Earn Dust + connect Ronin → unlock Axie NFTs as unique cards |
| **Competitive** | Climb ladder with starter + earned cards | Climb ladder + access Ranked Premium tier (NFT-gated for matchmaking quality, NOT power) |
| **Tournaments** | Free entry events (small prize pools) | AXS/SLP entry events (90 % players, 5 % burn, 5 % game treasury) — open to F2P with earned AXS |
| **Identity** | Username + avatar | Username + avatar + on-chain proof (wallet badge) |

**Both paths win.** Web2 players never feel locked out. Web3 players have real, mechanically meaningful utility for their NFTs.

## Why Sky Mavis benefits from Web 2.5 design

### Player base expansion
- **Web2 friction is eliminated** → 10x more potential players reach first session.
- The current Axie audience is ~50k DAU (Origins). Web2.5 onboarding could 10-50x this.

### NFT utility deepening
- Every Axie NFT becomes a playable card via deterministic algorithm.
- 3M+ Axies in circulation = 3M+ unique cards, no manual art needed.
- Demand for *specific parts* (not just "any Axie") creates Marketplace differentiation.

### Token sinks (deflationary pressure)
- Tournament entries split 5 % to permanent burn + 5 % to a transparent game treasury (10 % combined) per cycle.
- Pack openings burn 25% of AXS spent.
- Cosmetic permanent unlocks burn 100% of AXS.

### Counter-narrative to pay-to-win
- Top ladder reachable F2P (validated by tournament bracket data post-launch).
- NFT cards are **side-grades**, not upgrades.
- This directly addresses the 2021 backlash that hurt Axie Infinity's image.

### Risk mitigation
- Web2-only mode means the game survives even if regulatory/market headwinds slow Web3 adoption.
- Open source codebase means Sky Mavis can fork/inherit if the core team pivots — no platform lock-in risk.
